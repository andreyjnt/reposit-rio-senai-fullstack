## Pesquisa ##

1. O que é um array em programação? É uma lista que armazena elementos.

2. Por que é melhor usar um array do que criar várias variáveis separadas, como aluno1, aluno2
e aluno3? Para não ocupar tanta memoria, não ter mais trabalho e economizar tempo.

3. O que é índice em um array? É o numero da posição do elemento.

4. Por que o primeiro elemento de um array fica na posição 0? Por que é uma regra.

5. Para que serve o .length? Para saber o numero de elementos na array. 

6. Como acessamos o primeiro elemento de um array chamado alunos? console.log(alunos[0])

7. Como podemos alterar um valor dentro de um array? Alteramos acessando a posição (índice) desejada e atribuindo um novo valor

8. Qual a relação entre arrays e laços de repetição? Os laços de repetição são frequentemente usados para percorrer todos os elementos de um array, permitindo ler, exibir ou modificar seus valores.

9. Qual a diferença entre percorrer um array com for e apenas mostrar um item específico? Mostrar um item específico: acessa apenas uma posição do array.
Percorrer com for: acessa todos os elementos do array, um por um.

10. O que é um acumulador? Dê um exemplo de uso com notas ou valores. Um acumulador é uma variável usada para armazenar resultados parciais durante um processo repetitivo, como uma soma.

11. Para que servem operadores aritméticos como +, -, * e / dentro de um programa? Servem para realizar cálculos matemáticos.

+ → soma
- → subtração
* → multiplicação
/ → divisão

12. Para que servem operadores lógicos como && e ||? Servem para combinar condições.

&& (E): todas as condições devem ser verdadeiras.
|| (OU): pelo menos uma condição deve ser verdadeira.

13. Qual a diferença entre if/else e switch? if/else: usado para testar condições mais complexas.
switch: usado quando há várias opções para o mesmo valor.

14. O que os métodos push e pop fazem em um array? push(): adiciona um elemento ao final do array.
pop(): remove o último elemento do array. 

15. O que é uma matriz e como ela se relaciona com arrays? Uma matriz é um array de arrays, permitindo organizar dados em linhas e colunas.
