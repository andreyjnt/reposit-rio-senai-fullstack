# Perguntas 

- **1: O que é programação?** Programação é o processo de escrever instruções passo a passo para um computador executar tarefas.

- **2: Oque é um algoritmo?** é uma sequência de instruções lógicas, finitas e bem definidas, criada para resolver um problema ou executar uma tarefa específica.

- **3: Qual a diferença entre algoritmo e linguagem de programação?** Um algoritmo é a lógica, o raciocínio ou a "receita de bolo" que descreve os passos necessários para resolver um problema. A linguagem de programação é a ferramenta (como Python, Java ou C++) usada para traduzir esses passos em um código que o computador consegue ler e executar.

- **4: O que e JavaScript?** É uma linguagem de programação.

- **5: Onde o JavaScript pode ser ultilizado ?** Desenvolvimento Web (Frontend): É a linguagem padrão executada nos navegadores para criar sites dinâmicos, interações de usuário, animações e aplicações web complexas.

- **6: O que e o Node.js?** Ambiente de execução de javascript.

- **7: QuaI é a função do Node.js no desenvolvimento com JavaScript?** A principal função do Node.js é fornecer um ambiente de execução para que o JavaScript possa ser executado fora do navegador.

- **8: O que e o VS Code?** o VS Code é uma ferramenta para escrever, editar, executar e depurar código

- **9: QuaI a diferenc;a entre um editor de texto comum e um editor de c6digo?** A principal diferença é que um editor de código possui recursos específicos para auxiliar o desenvolvimento de software, enquanto um editor de texto comum é voltado apenas para criar e editar textos.

- **10: O que e o terminal?** o terminal é uma ferramenta que permite controlar o computador e executar tarefas por meio de comandos de texto, sendo muito utilizado por programadores para executar e gerenciar seus projetos.

- **11: Para que serve o comando node arquivo.js?** diz ao computador para executar o código JavaScript contido no arquivo especificado usando o Node.js.

- **12: O que e uma variavel?** Uma variável é um espaço na memória do computador usado para armazenar um valor que pode ser utilizado e, em muitos casos, alterado durante a execução de um programa.

- **13: QuaI a diferenc;a entre let e canst?** let → permite alterar (reatribuir) o valor da variável.
const → não permite reatribuir a variável após sua criação. Geralmente é a opção preferida quando o valor não precisa mudar.

- **14: O que sao tipos de dados? Cite exemplos** Tipos de dados são as diferentes formas de representar informações dentro de um programa, como números, textos, listas e valores lógicos.
