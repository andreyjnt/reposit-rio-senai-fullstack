## Perguntas 
O que é um objeto em programação?
É algo criado a partir de uma classe que guarda informações e pode executar ações.

O que é uma propriedade de um objeto? Cite um exemplo.
É uma característica do objeto. Exemplo: nome, idade ou cor.

O que é um método? Qual a diferença entre método e propriedade?
Método é uma função do objeto. A propriedade guarda um valor, e o método realiza uma ação.
O que é uma classe?

É um modelo usado para criar objetos.
Qual a diferença entre classe e objeto?
A classe é o modelo. O objeto é uma instância criada a partir desse modelo.

Para que serve o constructor dentro de uma classe?
Serve para inicializar o objeto com os valores quando ele é criado.

O que significa a palavra this dentro de uma classe?
this se refere ao próprio objeto que está usando a classe.

O que a palavra new faz ao criar um objeto?
Ela cria uma nova instância da classe e chama o constructor.

Em TypeScript, por que usamos tipos como string, number e boolean?
Para definir o tipo de cada dado e evitar erros no código.

Dê um exemplo do mundo real que poderia virar uma classe em código e cite 3 propriedades e 1 método.
Classe: Carro.
Propriedades: marca, modelo e ano.
Método: ligar().
