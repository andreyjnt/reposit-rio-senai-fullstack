## Perguntas ##
**1- O que é uma estrutura de repetição?**
É uma estrutura que permite executar um conjunto de instruções várias vezes.


**2- Para que serve o laço for?**
Serve para repetir um bloco de código quando a quantidade de repetições é conhecida.


**3- Para que serve o laço while?**
Serve para repetir um bloco de código enquanto uma condição for verdadeira.


**4- Para que serve o laço do-while?**
Serve para repetir um bloco de código, garantindo que ele seja executado pelo menos uma vez.


**3- Qual é a diferença entre for, while e do-while?**
O for é usado quando se sabe quantas repetições serão feitas, o while quando a repetição depende de uma condição, e o do-while executa o bloco ao menos uma vez antes de verificar a condição.
