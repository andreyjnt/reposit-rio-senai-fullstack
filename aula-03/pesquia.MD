## Perguntas ##

**1. O que é uma estrutura condicional em programação? Dê um exemplo do cotidiano usando "se... então... senão".**
É uma estrutura que permite ao programa tomar decisões de acordo com uma condição.

**2. Qual é a diferença entre usar apenas if e usar if/else?**
O **if** executa uma ação somente quando a condição é verdadeira. Já o **if/else** permite definir uma ação para quando a condição for verdadeira e outra para quando for falsa.

**3. Quando usamos else if? Crie um exemplo com três ou mais possibilidades.**
Usamos **else if** quando precisamos verificar várias condições diferentes e executar ações específicas para cada uma delas.

**4. O que significa uma condição retornar true ou false?**
Significa que a condição foi avaliada e o resultado foi verdadeiro (**true**) ou falso (**false**).

**5. Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?**
O **==** compara apenas os valores. O **===** compara os valores e também os tipos de dados. O mais recomendado é o **===**, pois evita comparações incorretas.

**6. Para que servem os operadores &&, || e !? Explique com um exemplo para cada um.**
O **&&** é usado quando todas as condições precisam ser verdadeiras. O **||** é usado quando pelo menos uma condição precisa ser verdadeira. O **!** serve para inverter o resultado lógico de uma condição.

**7. O que o operador % faz? Por que ele é útil para descobrir se um número é par ou ímpar?**
O operador **%** retorna o resto de uma divisão. Ele é útil porque permite verificar se a divisão por 2 deixa resto 0 ou não.

**8. O que é switch case? Em que situação ele fica melhor do que vários else if?**
O **switch case** é uma estrutura de seleção usada para escolher entre várias opções. Ele fica melhor do que vários **else if** quando há muitas comparações de um mesmo valor.

**9. Para que servem break e default dentro de um switch?**
O **break** interrompe a execução após encontrar o caso correto. O **default** define uma ação padrão quando nenhum caso é atendido.

**10. Por que é importante testar o mesmo programa com vários valores diferentes?**
Porque isso ajuda a verificar se o programa funciona corretamente em diferentes situações e a encontrar possíveis erros.
